# Linux Learning

A collection of Linux commands, concepts, troubleshooting notes, and shell scripts documented during my learning journey.

## About

This repository serves as my personal Linux knowledge base. I use it to document what I learn while exploring Linux, working with Fedora, using the command line, and writing shell scripts.

The goal is to learn Linux consistently, build practical skills, and maintain a record of my progress over time.

## Topics Covered

- Linux Commands
- File System and Directory Structure
- File Permissions and Ownership
- Process Management
- Package Management
- Shell Scripting
- System Administration Basics
- Networking Fundamentals
- Fedora Notes
- Troubleshooting and Fixes
